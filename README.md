# ForMyCutemm
献给我朋友的微信小程序

写这个小程序的初衷是想给我朋友一个生日惊喜，这个小程序配合上为他选择的礼物，我相信他会非常开心。

来看看我是如何操作的吧!
tip1:
制作下面这个小程序（稍后上图详述）
tip2:
在朋友生日前几天的时候为他网购一个礼物，并且随时留意物流信息
tip3:
当物流显示开始配送的时候，把这个小程序发给我朋友
tip4:
当我朋友做完小程序里最后一个选择题的时候，会从中得到一个提示：给他买的礼物快到了！
tip5:
时间预算准确的话，我朋友马上会收到物流的短信，就可以去拿他的礼物辣！
